**EduSmart - Dashboard Media Pembelajaran Interaktif**

**Deskripsi**
EduSmart adalah dashboard media pembelajaran berbasis web yang dikembangkan menggunakan HTML, CSS, dan JavaScript dengan framework Tailwind CSS. Dibuat sebagai proyek LKPD 12 mata pelajaran Front-End Development.

**Fitur**
- 🏠 Beranda dengan statistik progress dan leaderboard
- 📘 Materi pembelajaran interaktif dengan sistem XP
- 📝 Kuis interaktif dengan penilaian otomatis
- 🏆 Sistem badge dan level berdasarkan XP
- 📋 Evaluasi dengan perhitungan skor
- 🔐 Login UI
- 🌙 Dark Mode (tema gelap ungu)
- 📱 Responsive untuk mobile
- 🎨 Animasi hover dan transisi halaman

**Tools**
- HTML, CSS, JavaScript
- Git & GitHub
